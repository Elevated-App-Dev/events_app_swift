## Design Notes

### Color Palette (Suggested)
- **Primary**: #4A90D9 (Blue) - Buttons, highlights
- **Success**: #5CB85C (Green) - Positive changes, completed
- **Warning**: #F0AD4E (Orange) - Warnings, attention
- **Error**: #D9534F (Red) - Errors, losses
- **Background**: #1A1A2E (Dark) - Main background
- **Card**: #2D2D44 (Dark Gray) - Card backgrounds
- **Text**: #FFFFFF (White) - Primary text
- **Text Secondary**: #A0A0A0 (Gray) - Secondary text

### Typography
- **Headers**: Bold, 28-32pt
- **Body**: Regular, 18-22pt
- **Small**: Regular, 14-16pt
- **Buttons**: Bold, 20-24pt

### Touch Targets
- Minimum: 44x44 points
- Recommended: 48x48 points for primary actions
- Spacing: 8-16 points between interactive elements
